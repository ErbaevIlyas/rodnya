# 🧪 Гайд по тестированию оптимизации

## 🎯 Как тестировать

### Тест 1: Проверка индикатора соединения

**Шаги**:
1. Откройте приложение
2. Посмотрите на заголовок
3. Должен быть индикатор 🟢 Онлайн

**Ожидаемый результат**: Индикатор показывает 🟢 Онлайн

---

### Тест 2: Отправка сообщения при офлайне

**Шаги**:
1. Откройте приложение
2. Откройте DevTools (F12)
3. Перейдите на вкладку Network
4. Нажмите на "Offline" (слева внизу)
5. Напишите сообщение в чат
6. Нажмите "Send"

**Ожидаемый результат**:
- Сообщение не отправляется на сервер
- Показывается уведомление "Сообщение сохранено"
- Индикатор показывает 🔴 Офлайн (1 в очереди)

---

### Тест 3: Восстановление соединения

**Шаги**:
1. Продолжите с предыдущего теста (офлайн режим)
2. Напишите ещё 2-3 сообщения
3. Нажмите на "Offline" чтобы вернуться в онлайн
4. Подождите 2-3 секунды

**Ожидаемый результат**:
- Индикатор меняется на 🟢 Онлайн
- Все сообщения из очереди отправляются
- Очередь очищается
- Сообщения появляются в чате

---

### Тест 4: Загрузка файла при офлайне

**Шаги**:
1. Откройте приложение
2. Откройте DevTools (F12) → Network → Offline
3. Нажмите на кнопку "Прикрепить" (скpaperclip)
4. Выберите "Фото"
5. Выберите изображение
6. Добавьте подпись
7. Нажмите "Отправить"

**Ожидаемый результат**:
- Файл не загружается на сервер
- Показывается уведомление "Файл сохранен"
- Индикатор показывает 🔴 Офлайн (1 в очереди)

---

### Тест 5: Отправка файла при восстановлении

**Шаги**:
1. Продолжите с предыдущего теста (файл в очереди)
2. Нажмите на "Offline" чтобы вернуться в онлайн
3. Подождите 5-10 секунд

**Ожидаемый результат**:
- Файл загружается на сервер
- Файл появляется в чате
- Очередь очищается

---

### Тест 6: Проверка localStorage

**Шаги**:
1. Откройте приложение
2. Откройте DevTools (F12)
3. Перейдите на вкладку Console
4. Введите: `JSON.parse(localStorage.getItem('rodnya_message_queue'))`
5. Нажмите Enter

**Ожидаемый результат**:
- Если очередь пуста: `null`
- Если есть сообщения: массив объектов с типом, данными, временем

**Пример вывода**:
```javascript
[
  {
    id: 1234567890,
    type: "message",
    data: { message: "Привет" },
    timestamp: "2024-03-06T10:30:00.000Z",
    retries: 0
  }
]
```

---

### Тест 7: Проверка кеша

**Шаги**:
1. Откройте приложение
2. Откройте DevTools (F12)
3. Перейдите на вкладку Application
4. Нажмите на "Cache Storage" (слева)
5. Должны быть кеши: `rodnya-static-v15.3`, `rodnya-dynamic-v15.3`

**Ожидаемый результат**:
- Кеши существуют
- Содержат файлы (HTML, CSS, JS, изображения)

---

### Тест 8: Проверка скорости загрузки

**Шаги**:
1. Откройте приложение
2. Откройте DevTools (F12)
3. Перейдите на вкладку Network
4. Перезагрузите страницу (F5)
5. Посмотрите время загрузки

**Ожидаемый результат**:
- Первая загрузка: 2-5 сек (зависит от соединения)
- Вторая загрузка: <1 сек (из кеша)

---

### Тест 9: Проверка таймаутов API

**Шаги**:
1. Откройте приложение
2. Откройте DevTools (F12)
3. Перейдите на вкладку Network

6. Отправьте сообщение

**Ожидаемый результат**:
- Сообщение отправляется в течение 5 сек
- Если дольше - используется кеш
- Нет ошибок в консоли

---

### Тест 10: Проверка заглушек для изображений

**Шаги**:
1. Откройте приложение
2. Откройте DevTools (F12)
3. Перейдите на вкладку Network

**Ожидаемый результат**:

## 🔍 Проверка логов

### Где смотреть логи
1. Откройте DevTools (F12)
2. Перейдите на вкладку Console
3. Должны быть логи вида:
   - ✅ Подключено к серверу
   - ❌ Отключено от сервера
   - 📝 Сообщение добавлено в очередь
   - ⏳ Обработка очереди

### Примеры логов
```
✅ Подключено к серверу
🟢 Соединение восстановлено
📝 Сообщение добавлено в очередь: {id: 1234567890, type: "message", ...}
⏳ Обработка очереди (1 сообщений)
✅ Подключено к серверу
```

## 📊 Метрики для проверки

### Размер кеша
```javascript
// В консоли:
caches.keys().then(names => {
  names.forEach(name => {
    caches.open(name).then(cache => {
      cache.keys().then(requests => {
        console.log(`${name}: ${requests.length} файлов`);
      });
    });
  });
});
```

### Размер localStorage
```javascript
// В консоли:
let total = 0;
for (let key in localStorage) {
  total += localStorage[key].length;
}
console.log(`localStorage: ${(total / 1024).toFixed(2)} KB`);
```

### Время отправки сообщения
```javascript
// В консоли:
console.time('send-message');
// Отправьте сообщение
console.timeEnd('send-message');
```

## 🐛 Решение проблем при тестировании

### Проблема: Индикатор не показывается
**Решение**:
1. Перезагрузите страницу (F5)
2. Очистите кеш: F12 → Application → Clear storage
3. Проверьте консоль на ошибки

### Проблема: Сообщения не сохраняются
**Решение**:
1. Проверьте localStorage: `localStorage.getItem('rodnya_message_queue')`
2. Проверьте консоль на ошибки
3. Убедитесь что вы в офлайн режиме

### Проблема: Сообщения не отправляются при восстановлении
**Решение**:
1. Проверьте соединение
2. Перезагрузите приложение
3. Проверьте консоль на ошибки

### Проблема: Кеш не создаётся
**Решение**:
1. Проверьте что Service Worker зарегистрирован
2. Откройте DevTools → Application → Service Workers
3. Должен быть Service Worker со статусом "activated"
