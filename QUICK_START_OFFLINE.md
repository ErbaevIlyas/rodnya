# ⚡ Быстрый старт: Работа при плохом соединении

## 🎯 За 30 секунд

Приложение **автоматически** работает при плохом соединении. Ничего не нужно настраивать!

## 🟢 Что изменилось

### 1. Индикатор соединения
В заголовке приложения:
- 🟢 **Онлайн** - соединение есть
- 🔴 **Офлайн** - нет соединения
- Показывает количество сообщений в очереди

### 2. Очередь сообщений
Если нет соединения:
1. Вы пишете сообщение
2. Оно сохраняется локально
3. Вы видите уведомление
4. При восстановлении соединения сообщение отправляется автоматически

### 3. Кеширование
- Приложение кеширует все файлы
- Работает быстрее даже при хорошем соединении
- Можно открыть приложение без интернета (если были на нём раньше)

## 📱 Примеры использования

### Пример 1: Метро
```
1. Вы в метро (нет соединения)
2. Пишете сообщения - они сохраняются
3. Выходите из метро
4. Все сообщения отправляются автоматически
```

### Пример 2: Слабый сигнал
```
1. Вы в месте со слабым сигналом
2. Загружаете файл
3. Файл сохраняется в очередь
4. При восстановлении сигнала файл отправляется
```

### Пример 3: Переключение сетей
```
1. Вы переключаетесь с WiFi на мобильную сеть
2. Может быть кратковременный разрыв
3. Сообщения сохраняются
4. При восстановлении отправляются
```

## 🔧 Как проверить

### Проверить индикатор
1. Откройте приложение
2. Посмотрите на заголовок
3. Должен быть индикатор 🟢 Онлайн

### Проверить очередь
1. Откройте DevTools (F12)
2. Перейдите на вкладку Console
3. Введите: `JSON.parse(localStorage.getItem('rodnya_message_queue'))`
4. Должен показать массив сообщений (или null если пусто)

### Проверить кеш
1. Откройте DevTools (F12)
2. Перейдите на вкладку Application
3. Нажмите на Cache Storage
4. Должны быть кеши: rodnya-static-v15.3, rodnya-dynamic-v15.3

## 🐛 Если что-то не работает

### Сообщение не отправилось
1. Проверьте индикатор соединения
2. Перезагрузите страницу (F5)
3. Проверьте консоль (F12 → Console)

### Очередь не очищается
1. Проверьте соединение
2. Перезагрузите приложение
3. Очистите localStorage: `localStorage.clear()`

### Приложение работает медленно
1. Очистите кеш: F12 → Application → Clear storage
2. Перезагрузите страницу
3. Проверьте скорость интернета

## 📊 Статистика

- **Размер кеша**: ~500KB
- **Время загрузки**: <1 сек (из кеша)
- **Максимум в очереди**: неограниченно
- **Время отправки очереди**: <5 сек
